I would like to have the Statistical Calculator tested:

  1. Open Postman. If you haven't installed it yet, you can download it from the official website.
  2. In Postman, create a new request tab and set the method to POST.
  3. Enter the following URL in the address bar (replace "SECRET_IP_GIVEN_TO_YOU" with the IP address provided to you): http://YOUR_SECRET_IP:8080/numbers
  4. Below the URL bar, go to the Body tab.
  5. Select the raw option and from the dropdown on the right, choose JSON.
  6. In the text area that appears, you can input one of the following JSON structures:

    7. Example 1:

    
{
    "numbers": [1, 2, 3, 4, 5],
    "is_population": false
}

    Example 2:

    
{
    "numbers": [-5, 10.3,15.289, 20.161, -225.88, 30.42],
    "is_population": true
}
  7. Click on the Send button to send the request.
  8. You should receive a basic statistical calculations of the numbers you have sent.
